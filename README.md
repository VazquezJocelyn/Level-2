# Level 2: Us  
### A 2nd Anniversary RPG

A custom-built pixel RPG created to celebrate two years of love.

Instead of flowers, I built a world.

•·················•·················•·················•

## 🎮 About

Level 2: Us is a browser-based RPG built with Phaser 3.

It features:
- Custom pixel sprite sheets  
- Interactive objects  
- A handcrafted map  
- Anniversary moments hidden throughout the game  

•·················•·················•·················•

## 🛠 Built With

- JavaScript
